# Forge OSS launch kit

This folder is the source of truth for the public launch campaign.

- `POSITIONING.md` - audience, category, message hierarchy, claims, objections
- `LAUNCH_COPY.md` - GitHub, Hacker News, Reddit, LinkedIn, X, newsletter, Product Hunt
- `DEMO_SCRIPT.md` - 90-second product demo and shot list
- `PRESS_BRIEF.md` - concise media and podcast briefing
- `CASE_STUDY_TEMPLATE.md` - evidence-first customer/project story template
- `COMMUNITY_PLAN.md` - launch sequence, channels, cadence, metrics
- `RELEASE_CHECKLIST.md` - hard launch gates and day-of checklist
- `SOCIAL_CAMPAIGN.md` - social copy, 30-second storyboard, voiceover, and publishing notes
- `landing/index.html` - standalone launch landing page using the console aesthetic
- `assets/forge-og.svg` - editable 1200x630 social card
- `assets/forge-campaign-30s.mp4` - rendered 9:16 social campaign master

## Core message

**Forge is the reliability harness for long-running coding agents.** It adds durable recovery, deterministic orchestration, independent test verification, and an operator control plane to local or OpenAI-compatible models.

Do not market Forge as a secure host sandbox or as a replacement for human review. Model-proposed commands must run in a container or VM unless an operator explicitly accepts host execution risk.
