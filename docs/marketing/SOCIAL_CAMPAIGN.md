# Forge social launch campaign

## Campaign line

**Agents act. Forge proves.**

## Primary post

Coding agents can write code. But can they recover from failure, show their work, and prove the tests passed?

Forge is an open-source reliability harness for long-running coding agents:

- durable Postgres checkpoints
- Planner -> Auditor -> Executor -> Evaluator
- independent test verification
- local or cloud OpenAI-compatible models
- an operator console for evidence and approvals

The model proposes. Forge keeps the state and checks the proof.

Explore the project: https://github.com/sciencenerd-des/forge
