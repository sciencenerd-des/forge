# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and personal
attacks are not tolerated. Maintainers may remove comments, commits, and
contributions that violate this code, and may ban repeat offenders.

Report unacceptable behavior to the maintainers via a GitHub Security advisory or a
direct message to a maintainer. Reports are handled confidentially.
